Setup:
```kotlin
private val classDir = File("../dsl-runtime/build/classes/kotlin/test")
private lateinit var compilationResult: KotlinCompilation.Result
private lateinit var ctx: DSLContext

@BeforeAll
fun setup() {
    // Create Postgres schema
    ctx = DSL.using(postgres.jdbcUrl, postgres.username, postgres.password)
    ctx.execute("CREATE TABLE organization (id BIGSERIAL PRIMARY KEY, name VARCHAR(255) NOT NULL)")
    ctx.execute("CREATE TABLE app_user (id BIGSERIAL PRIMARY KEY, name VARCHAR(255) NOT NULL, email VARCHAR(255) NOT NULL, organization_id BIGINT NOT NULL REFERENCES organization(id))")
    ctx.execute("CREATE TABLE category (id BIGSERIAL PRIMARY KEY, name VARCHAR(255) NOT NULL, parent_id BIGINT REFERENCES category(id))")
    ctx.execute("CREATE TABLE task (id BIGSERIAL PRIMARY KEY, title VARCHAR(255) NOT NULL, created_by BIGINT NOT NULL REFERENCES app_user(id), updated_by BIGINT REFERENCES app_user(id))")

    // Generate DSL sources
    val generatedSources = CodeGenerator().generateSource(classDir, "com.example.generated", "com.example.declarativejooq")

    // Compile with TestHarness
    val sources = generatedSources.map { (name, code) -> SourceFile.kotlin(name, code) } + listOf(integrationHarnessSource())
    compilationResult = KotlinCompilation().apply {
        this.sources = sources
        inheritClassPath = true
        messageOutputStream = System.out
        kotlincArguments = listOf("-Xskip-metadata-version-check")
        classpaths = classpaths + listOf(classDir)
    }.compile()
    assertEquals(KotlinCompilation.ExitCode.OK, compilationResult.exitCode, "Compilation failed:\n${compilationResult.messages}")
}
```

**@BeforeEach**: Truncate all tables in reverse FK order (task, app_user, category, organization) using `TRUNCATE ... CASCADE`.

**integrationHarnessSource()**: A SourceFile.kotlin with an object containing methods:
- `runBasic(ctx)`: one org + one user (root + nested)
- `runMultipleSameType(ctx)`: one org + two users
- `runMultiLevel(ctx)`: org + user (two levels deep)
- `runSelfRef(ctx)`: category -> childCategory -> childCategory (3 levels)
- `runMultiFk(ctx)`: org + user + task nested under createdBy
- `runMixedGraph(ctx)`: combination of all: org with users, category tree, task

**Test methods** (each calls harness via reflection, then asserts DB state):

1. **`rootAndNestedRecords`**: runBasic -> 1 org, 1 user, user.organization_id = org.id
2. **`multipleSameTypeRecords`**: runMultipleSameType -> 1 org, 2 users, both users have organization_id = org.id
3. **`multiLevelNesting`**: runMultiLevel -> verify three-level chain
4. **`selfReferentialFkTwoPassInsert`**: runSelfRef -> 3 categories, correct parent_id chain, root has NULL parent_id
5. **`multipleFksToSameTable`**: runMultiFk -> task.created_by = user.id, task.updated_by IS NULL
6. **`mixedGraph`**: runMixedGraph -> all tables populated, all FKs correct

Each test queries the Postgres database directly with jOOQ DSL to assert FK values.
