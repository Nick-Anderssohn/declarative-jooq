1. Create `ClasspathScanner.kt` in `codegen/src/main/kotlin/com/example/declarativejooq/codegen/scanner/`:
   - Class with two methods: `findTableClassNames(classDir: File, packageFilter: String? = null): List<String>` and `findRecordClassNames(classDir: File, packageFilter: String? = null): List<String>`
   - `findTableClassNames` uses `ClassGraph().overrideClasspath(classDir.absolutePath).enableClassInfo()`, applies `.acceptPackages(packageFilter)` if non-null, calls `.scan()`, then `scan.getSubclasses("org.jooq.impl.TableImpl").filterNot { it.isAbstract }.map { it.name }`
   - `findRecordClassNames` same pattern but `scan.getSubclasses("org.jooq.impl.UpdatableRecordImpl").filterNot { it.isAbstract }.map { it.name }`
   - Close the ScanResult in a `use` block

2. Create `MetadataExtractor.kt` in `codegen/src/main/kotlin/com/example/declarativejooq/codegen/scanner/`:
   - Class with method `extract(classDir: File, tableClassNames: List<String>): List<TableIR>`
   - Creates a `URLClassLoader(arrayOf(classDir.toURI().toURL()), Thread.currentThread().contextClassLoader)` — parent classloader is the thread context classloader so jOOQ types resolve
   - For each table class name: loads the class via the URLClassLoader, calls `loadTableInstance(klass)` to get the TableImpl singleton
   - `loadTableInstance(klass: Class<*>): org.jooq.impl.TableImpl<*>` — scans `klass.declaredFields` for static fields (`java.lang.reflect.Modifier.isStatic(it.modifiers)`) whose type equals `klass`, gets the first one, casts to `TableImpl<*>`. Does NOT look for a field named "INSTANCE".
   - Extracts the `tableConstantName` from the field name that held the table singleton (e.g., "ORGANIZATION", "APP_USER")
   - For columns: iterates `tableInstance.fields()`, for each field creates a `ColumnIR` with:
     - `columnName` = `field.name` (the SQL column name, e.g., "id", "name")
     - `propertyName` = snake_case to camelCase conversion: split on `_`, first segment lowercase, subsequent segments capitalized first char, join. E.g., "organization_id" -> "organizationId"
     - `kotlinTypeName` = map from `field.dataType.type` (Java Class) to KotlinPoet TypeName:
       - `java.lang.Long` / `Long::class.java` -> `com.squareup.kotlinpoet.LONG`
       - `java.lang.Integer` / `Int::class.java` -> `com.squareup.kotlinpoet.INT`
       - `java.lang.String` -> `com.squareup.kotlinpoet.STRING`
       - `java.lang.Boolean` / `Boolean::class.java` -> `com.squareup.kotlinpoet.BOOLEAN`
       - `java.math.BigDecimal` -> `ClassName("java.math", "BigDecimal")`
       - `java.time.LocalDate` -> `ClassName("java.time", "LocalDate")`
       - `java.time.LocalDateTime` -> `ClassName("java.time", "LocalDateTime")`
       - else -> `field.dataType.type.asTypeName()`
     - `isIdentity` = true if `tableInstance.identity?.field == field`
     - `isNullable` = `field.dataType.nullable()`
     - `tableFieldRefExpression` = `"${tableClassName}.${tableConstantName}.${field.name.uppercase()}"` — but use the actual field name from the TableField (accessed via reflection on the table class: find the declared field on the table class whose get() value equals this TableField)
   - For FKs: iterates `tableInstance.references`, for each FK where `fk.fields.size == 1` (skip multi-column), creates a `ForeignKeyIR`:
     - `fkName` = `fk.name`
     - `childTableName` = current table's tableName
     - `childFieldExpression` = the tableFieldRefExpression of the FK column (look up in columns list)
     - `parentTableName` = `fk.key.table.name` (the referenced table's SQL name)
     - `parentBuilderClassName` = derived from parent table name: snake_case to PascalCase + "Builder"
     - `builderFunctionName` = derived from child table name: snake_case to camelCase (singular). E.g., "app_user" -> "appUser". For the child builder function INSIDE the parent, use the child table's camelCase name.
   - After all tables are extracted: cross-link inboundFKs. For each table's outboundFKs, find the parent TableIR by tableName and add the FK to its `inboundFKs` list.
   - Set `isRoot = outboundFKs.isEmpty()` on each TableIR
   - `builderClassName` = PascalCase(tableName) + "Builder" (e.g., "organization" -> "OrganizationBuilder")
   - `resultClassName` = PascalCase(tableName) + "Result" (e.g., "organization" -> "OrganizationResult")
   - `dslFunctionName` = camelCase(tableName) (e.g., "organization" -> "organization", "app_user" -> "appUser")
   - `tableClassName` = the simple name of the loaded class (e.g., "OrganizationTable")
   - `recordClassName` = obtained from `tableInstance.recordType.simpleName` (e.g., "OrganizationRecord")
   - `sourcePackage` = the package of the loaded table class
