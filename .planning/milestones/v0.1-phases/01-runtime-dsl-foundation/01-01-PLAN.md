Create the following source files in `dsl-runtime/src/main/kotlin/com/example/declarativejooq/`:

**DeclarativeJooqDsl.kt** -- the @DslMarker annotation:
```kotlin
package com.example.declarativejooq

@DslMarker
@Target(AnnotationTarget.CLASS)
annotation class DeclarativeJooqDsl
```

**RecordNode.kt** -- internal representation of a single declared record:
```kotlin
package com.example.declarativejooq

import org.jooq.Field
import org.jooq.Table
import org.jooq.TableField
import org.jooq.UpdatableRecord

class RecordNode(
    val table: Table<*>,
    val record: UpdatableRecord<*>,
    val parentNode: RecordNode?,
    val parentFkField: TableField<*, *>?,
    val declarationIndex: Int
) {
    val children: MutableList<RecordNode> = mutableListOf()
}
```
- `parentNode` is null for root records
- `parentFkField` is the FK column on THIS record's table that points to the parent's PK
- `declarationIndex` is a monotonically increasing integer assigned during DSL block execution to preserve declaration order (DSL-08)
- `record` is the actual jOOQ UpdatableRecord that will be inserted

**RecordGraph.kt** -- ordered collection of all record nodes:
```kotlin
package com.example.declarativejooq

class RecordGraph {
    private val _rootNodes: MutableList<RecordNode> = mutableListOf()
    val rootNodes: List<RecordNode> get() = _rootNodes

    private var nextIndex = 0

    fun nextDeclarationIndex(): Int = nextIndex++

    fun addRootNode(node: RecordNode) {
        _rootNodes.add(node)
    }

    fun allNodes(): List<RecordNode> {
        val result = mutableListOf<RecordNode>()
        fun collect(node: RecordNode) {
            result.add(node)
            node.children.forEach { collect(it) }
        }
        _rootNodes.forEach { collect(it) }
        return result
    }
}
```

**RecordBuilder.kt** -- abstract base for table-specific builders:
```kotlin
package com.example.declarativejooq

import org.jooq.Table
import org.jooq.TableField
import org.jooq.UpdatableRecord

@DeclarativeJooqDsl
abstract class RecordBuilder<R : UpdatableRecord<R>>(
    val table: Table<R>,
    val parentNode: RecordNode?,
    val parentFkField: TableField<*, *>?,
    val recordGraph: RecordGraph
) {
    abstract fun buildRecord(): R

    fun build(): RecordNode {
        val record = buildRecord()
        val node = RecordNode(
            table = table,
            record = record,
            parentNode = parentNode,
            parentFkField = parentFkField,
            declarationIndex = recordGraph.nextDeclarationIndex()
        )
        parentNode?.children?.add(node)
        return node
    }
}
```
- Subclasses (generated or hand-written) implement `buildRecord()` to create a jOOQ record with user-set field values
- `build()` wraps the record in a `RecordNode` with parent context

**DslResult.kt** -- result container:
```kotlin
package com.example.declarativejooq

import org.jooq.UpdatableRecord

class DslResult(
    private val recordsByTable: LinkedHashMap<String, MutableList<UpdatableRecord<*>>>
) {
    fun <R : UpdatableRecord<R>> records(tableName: String): List<UpdatableRecord<*>> {
        return recordsByTable[tableName] ?: emptyList()
    }

    fun allRecords(): Map<String, List<UpdatableRecord<*>>> {
        return recordsByTable
    }
}
```
- Uses LinkedHashMap to preserve table insertion order (DSL-08)
- Phase 1 uses generic access by table name; Phase 2 codegen will produce typed wrapper classes
